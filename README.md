## 환경구축


node.js 설치<br>
npm 설치

news/package.json 에 있는 dependency 설치<p>
    news.zip 압축 풀기<br>
    cd news/<br>
    npm install<br>
    약 10~20초 소요<br>





## 배포


배포에 사용할 포트는 news/main.js 15번 줄에서 변경가능하다.


npm start<p>
브라우저에서 ip: 설정한 포트 입력하여 배포 확인<br>

*백그라운드 실행 시 (리눅스):<br>
	$ sudo nohup npm start &
